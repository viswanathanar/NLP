1) <TaskIt.R> script is the main routine which reads the input file <Input Tasks> and generates an Output file <Results>. TaskIt.R calls an R source <Auxiliary Functions.R> 

2)<Auxiliary Functions.R> is source repository for several functions like entity detection, POS Tagging and a function to break the POS tags in to 4 major categories namely Noun, Verb, Adverb and Adjective.
